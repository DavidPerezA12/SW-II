## Temática
API REST para gestionar una base de datos de videojuegos, plataformas y reseñas.

La API permitirá:
- Consultar videojuegos
- Filtrar por plataforma o género
- Añadir reseñas
- Obtener información externa de videojuegos mediante la API RAWG.
